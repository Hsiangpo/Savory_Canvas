# 错误码映射表

| 错误码 | 含义 | 默认中文消息 | 典型触发场景 |
|--------|------|--------------|--------------|
| E-1001 | 视频转写失败 | 视频转写失败 | 转写 worker 执行异常 |
| E-1002 | 风格对话结构异常 | 风格对话返回结构异常 | 上游模型返回非预期结构 |
| E-1003 | 资产不足 | 素材不足，无法生成 | 生成任务无可用素材 |
| E-1004 | 生图部分失败 | 部分图片生成失败 | 多图任务中仅部分成功 |
| E-1005 | 导出失败 | 导出流程异常 | 导出文件生成失败 |
| E-1006 | 模型配置缺失或不可用 | 请先完成模型设置 | 未配置模型路由或 provider 不可用 |
| E-2001 | 会话不存在 | 会话不存在 | 查询/引用不存在的会话 |
| E-2002 | 素材不存在 | 素材不存在 | 查询不存在的素材或转写 |
| E-2003 | 风格不存在 | 风格不存在 | 更新/删除不存在的风格 |
| E-2004 | 任务不存在 | 任务不存在 | 查询/取消不存在的生成任务 |
| E-2005 | 导出任务不存在 | 导出任务不存在 | 查询不存在的导出任务 |
| E-2006 | 提供商不存在 | 提供商不存在 | 更新/删除不存在的提供商 |
| E-1099 | 未知内部错误 | 系统内部错误 | 未归类异常 |

实现位置：
- 全局异常：`backend/app/main.py`
- 任务失败码：`backend/app/workers/generation_worker.py`
- 导出失败码：`backend/app/workers/export_worker.py`
- 转写失败码：`backend/app/workers/transcript_worker.py`
- 模型配置错误：`backend/app/services/model_service.py`
